# ほやほやほんやく (hoyaTranslate)

ブラウザ上の日本語テキストを「ほや語」に変換する Chrome 拡張機能。

- ランディング: https://wassyoi-fes.github.io/hoyaTranslate/
- プライバシーポリシー: https://wassyoi-fes.github.io/hoyaTranslate/privacy.html
