概要

「ほやほやほんやく」(以下「本拡張機能」)は、ブラウザに表示されているテキストを「ほや語」に変換する Chrome 拡張機能です。本ポリシーは、本拡張機能の利用にともなうデータの取扱いについて定めるものです。

収集・送信する情報

本拡張機能は、利用者の個人情報・閲覧履歴・ページ内容・その他あらゆる情報を、外部サーバーや第三者に送信・収集しません。テキストの解析および置換はすべて利用者のブラウザ内で完結します。

ローカルに保存される情報

以下の利用者設定を、Chrome の storage.local API を用いて利用者のブラウザ内にのみ保存します。

  • ほんやくの ON / OFF 状態
  • 選択中のモード
  • 解放済みモードの記録

これらの情報は外部に送信されません。本拡張機能をブラウザからアンインストールした時点で、上記情報は削除されます。

権限の利用理由

本拡張機能は以下の Chrome 権限を使用します。

host_permissions: <all_urls>
利用者が閲覧中の Web ページ上のテキストを「ほや語」に変換するために必要です。本拡張機能はユーザーが訪問するあらゆる Web ページを対象とするため、特定のドメインに限定することができません。ページ内容の読み取りおよび表示テキストの置き換えという目的のみに使用され、読み取った内容が外部に送信されることはありません。
storage
上記「ローカルに保存される情報」を利用者のブラウザ内に保存するために必要です。

ページ内容の取扱い

本拡張機能はページ上のテキストを読み取り、ブラウザ内のメモリ上でのみ「ほや語」への変換処理を行い、その結果に置き換えます。読み取った内容が外部に送信されることはありません。

第三者への提供

本拡張機能は、利用者から取得したいかなる情報も第三者に提供することはありません。

本ポリシーの変更

本ポリシーの内容は、必要に応じて変更されることがあります。重要な変更があった場合は本ページにて告知します。
