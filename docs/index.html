ほやほやほんやく

ブラウザ上の日本語テキストを「ほや語」に変換する Chrome 拡張機能

これは何?

あらゆる Web ページの日本語テキストを、ほやドル界隈で使われている「ほや語」に置き換える Chrome 拡張機能です。挨拶や定型表現、関連する漢字熟語をまとめて変換し、いつものブラウジングをほやっほーな雰囲気に変えます。

4つのモード

ほやモード
挨拶や定型表現を中心に「ほや語」化します。
スーパーほやモード
関連する漢字熟語も加えて「ほや語」化します。
ウルトラほやモード
対応する全表現を「ほや語」化します。
ウルトラほや逆ほんやくモード
ほや語っぽく見える文章を、なるべく元の言葉に戻して読みます。
