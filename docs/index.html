Chrome Extension

Web を、
ほや語に。

ほやドル公認の Chrome 拡張機能。
ブラウザに映る日本語を、まるごと「ほや語」に変換します。

About

これは何?

あらゆる Web ページの日本語テキストを、ほやドルが日頃使っている「ほや語」に置き換える Chrome 拡張機能です。挨拶や定型表現、関連する漢字熟語をまとめて変換し、いつものブラウジングをほやっほーな雰囲気に変えます。
